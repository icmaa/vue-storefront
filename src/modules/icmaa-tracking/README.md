# `icmaa-tracking` module

Get order tracking information from API.

## Configs

```json
...
{
  "icmaa_tracking": {
    // Endpoint of the api
    "endpoint": "/api/ext/icmaa-tracking",
    // The tracking-link is only shown for orders in this status
    "orderStatusWhitelist": ["complete"]
  }
}
```
